It should help you to find similar photos in your collection and try to apply the techniques from the video - ideal for learning new stuff.

This repository should help you to find what type of images is being edited / corrected in particular tutorial/video without watching all of them.
Photos shown in videos are one place, sorted alphabetically with few details.
